# Java File I/O - Notes App

A simple command-line Notes Manager developed in Java that allows users to save and read notes using file handling.

## Features
- Add notes to a text file
- View all saved notes
- Stores data permanently using File I/O
- Handles file exceptions using try-catch

## Technologies Used
- Java
- FileReader
- FileWriter
- BufferedReader
- Exception Handling

## Project Structure

```
NotesApp.java
notes.txt
README.md
```

## How to Run

1. Compile the program:
   ```
   javac NotesApp.java
   ```

2. Run the program:
   ```
   java NotesApp
   ```

## Sample Menu

```
===== Notes Manager =====
1. Add Note
2. View Notes
3. Exit
```

## Concepts Covered
- File I/O
- FileReader
- FileWriter
- BufferedReader
- Exception Handling
- Try-with-resources

## Outcome
This project demonstrates how to perform file reading and writing in Java while implementing a simple text-based notes management system.
